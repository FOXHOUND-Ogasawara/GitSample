Q2
以下のアルゴリズムは、数値の配列をソートするためのアルゴリズムです。
手順通りに実装をして同じ出力結果にしてください。

Arraysライブラリのメソッドを使用するため java.util.Arrays をインポートしておいてください

1. 以下の配列を宣言する
    int[] array = {7, 3, 10, 5, 1}
2. 2重ループを使用します
2-1. 外側のループは変数名をouterとし 0 から arrayの要素数 - 1 未満まで1ずつ増やしながら繰り返します。
2-2. 内側のループは変数名をiとし arrayの要素数 - 1 から outer まで1ずつ減少させながら繰り返します。
3. 内側のループの処理は下記の通りです
3-1. array[i] と array[i - 1]を比較し、array[i] がarray[i - 1]より小さい場合以下の処理をする
3-2. 変数 tmp に array[i]を代入する
3-3. array[i] に array[i - 1]を代入する
3-4. array[i - 1] に tmp を代入する
4. 外側のループ、内側のループが全て終わった後、以下の構文を出力する
Arrays.toString(array)

出力結果）
[1, 3, 5, 7, 10]
